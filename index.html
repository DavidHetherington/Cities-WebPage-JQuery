Welcome to the Minimal SaaS demonstration architecture Cities demonstration. This demonstration consists of three pieces.

  • Web Front End - This webpage connects to the back end (business logic) server using a RESTful API.
  • Back End - Provides the RESTful API and connects to a MongoDB NoSQL database instance. The back end is implemented in Node.js and Express and hosted on an Amazon EC2 Ubuntu instance.
  • Database - The database is hosted by Mongolab.com and is a MongoDB NoSQL document database.

Note: the add, delete, update, and column sort functions are not implemented yet.
